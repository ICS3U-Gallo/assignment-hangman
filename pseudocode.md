# Hangman (Pseudocode)

```
1. P1 Chooses a word or series of words
2. P2 Chooses letter or word
3. if P1 chooses a letter and the letter is in the word, reveal all spots where it is found
4. elif player chooses a letter and letter is not in the word, add a body part to the hangman
5. elif the player chooses the right word and the hangman isn't finished yet, he/she wins the game
6. elif the player chooses the wrong word, add a body part
7. If the hangman is finished, game ends and P1 wins, P2 loses
8. Repeat steps 2-7 until game ends
```
